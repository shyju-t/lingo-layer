# Screenshot Guide for Chrome Web Store

Chrome Web Store requires at least 1 screenshot at **1280x800 px** (up to 5).

## How to capture

1. Resize your Chrome window to **1280x800** pixels:
   - Open DevTools (F12) > click the three-dot menu > "More tools" > "Device Mode"
   - Or use a window resizer extension
2. Navigate to a Reddit post with LingoLayer active
3. Take screenshots using Cmd+Shift+4 (Mac) or a screenshot tool

## Recommended screenshots

1. **Injection in action** -- Reddit post with German phrases injected, one tooltip visible showing grammar info and speaker icon
2. **Popup settings** -- The extension popup with the Learning section open, showing language and CEFR level dropdowns
3. **Provider selection** -- The popup with the LLM Provider section expanded, showing the provider dropdown and model selector
4. **Voice feature** -- A phrase with the speaker icon visible/highlighted
5. **Multi-language** -- Switch to a different language (e.g. Spanish or French) to show versatility

## File naming

Save as PNG at 1280x800 in this folder:
- `screenshot-1-injection.png`
- `screenshot-2-popup.png`
- `screenshot-3-provider.png`
- `screenshot-4-voice.png`
- `screenshot-5-multilang.png`
